▶ <b>유인물</b>  
- 첫 번째 유인물의 <b> QR코드를 스캔하시면 프로젝트의 설명이 담긴 PPT 링크로 연결</b>됩니다.
- 사용된 도면은 '선문대학교 인문관' 입니다.
- 핸드폰 번호는 개인정보라서 삭제했습니다.
